# 🔍 Investigação Genealógica — Família Brandi de Minas Gerais (Origem: Baependi)

> **Objetivo principal:** Encontrar o registro de **batismo de Maria Guilhermina Brandi** (nascida ~1862–1865, provavelmente em Baependi, MG), filha provável do imigrante italiano Nicola Brandi.

Este repositório contém os arquivos de pesquisa e documentação genealógica da Família Brandi (estruturados a partir de arquivos `.md` do Obsidian). Incluímos aqui todas as ramificações conhecidas da família, irmãos, tios e descendentes associados.

## 🚨 Status da Investigação
- ✅ **Casamento dos pais (Nicola + Anna Silvéria, 1861)** — Confirmado.
- ✅ **Filhos de Maria Guilhermina (1879–1898)** — 12 registros de descendentes diretos encontrados.
- ⚠️ **Casamento de Maria Guilhermina com Paulino** — Não localizado.
- 🔴 **Batismo de Maria Guilhermina** — **Em busca**.
- ⚠️ **Óbito de Maria Guilhermina** — Não localizado.
